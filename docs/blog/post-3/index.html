まとめ

Dockerは自分のpc環境を汚さずに開発出来るだけでなく、
開発環境を高速に構築出来る最高のツールだと考えています。
しかし、イメージのサイズやキャッシュ/ビルドなどを考えずに
利用していると、pcのディスク容量が逼迫していきます。
ビルド時間も高速化を検討しないと、
開発する時間がとれなくなってしまいます。
Dockerを便利に利用するために、
これからもベストプラクティスを考えていこうと思います。
